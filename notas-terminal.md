Mis Notas de Terminal - TP1
Comandos utilizados en el desarrollo del trabajo práctico.
1) pwd: Muestra la ruta de la carpeta actual,lo usé para saber si estaba en la carpeta de mi proyecto.
2) ls: Lista los archivos y carpetas. Lo usé para ver si mi archivo "index.html" ya estaba creado.
3) cd: Cambia de directorio. Lo usé para entrar a la carpeta `tp1-mi-sitio`.
4) mkdir: Crea una carpeta nueva. Lo usé para crear la carpeta `images`.
5) touch: Crea archivos vacíos. Lo usé para crear `style.css` y `notas-terminal.md`.
6) cat: Muestra el contenido de un archivo en la terminal. Lo usé para revisar mi README.
7) which: Muestra la ubicación de un ejecutable. Lo usé para encontrar la ruta de instalación de Git .
8) code .: Abre la carpeta actual en Visual Studio Code. Lo usé para comenzar a realizar el trabajo.
9) git --version: Verifica la versión de Git instalada. Lo usé para la captura de pantalla del Punto 1.
10) git config --list: Muestra la configuración de Git. Lo usé para registrar mis datos y despues verificar que mi nombre y gmail estuvieran bien puestos.
